# Portfolio | Álex Pascual Hernández


Hola buenas, soy Alejandro Pascual Hernández, graduado en __Biología__, en el máster de __Mejora Genética Vegetal__ y actualmente estudiando el máster de __Bioinformática__ de la UNIR. 

### Breve descripción y objetivos 

Este repositorio de GitHub se trata de un proyecto dentro del máster de bioinformática en el cual se nos insta a desarrollar un portfolio personal. Esto tiene como objetivo no solo familiarizarnos con el entorno de GitHub, lo cual es imprescindible en el mundo de la bioinformática, sino también exponer nuestros avances dentro y fuera del máster. 
En este porfolio podréis ver todos los trabajos de clase divididos por asignaturas, trabajos personales y trabajos realizados en mis prácticas extracurriculares. En todo momento habrá enlaces al repositorio de GitHub en el que se podrá observar la estructura del porfolio. 

### Aclaraciones pertinentes

El código está escrito en HTML, CSS y JavaScript y estará en continuo desarrollo, mejorando a medida que mis conocimientos en programación aumenten a lo largo del máster.
